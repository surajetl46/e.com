# E-commerce Recommendation System

This project builds a recommendation system for an e-commerce platform using collaborative and content-based filtering techniques.

## File Structure
- `app/`: Core functionality for data processing, modeling, and recommendation logic.
- `scripts/`: Scripts for running training and starting the Flask server.
- `tests/`: Unit tests for different modules.
- `data/`: Folder containing the dataset.
